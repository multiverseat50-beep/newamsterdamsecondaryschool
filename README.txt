NEW AMSTERDAM SECONDARY SCHOOL WEBSITE

UPLOAD TO GITHUB
1. Extract this ZIP file.
2. Upload all files to the main level of the existing GitHub repository.
3. Keep the CNAME file in the repository.
4. Commit the changes.

ADMISSIONS LINKS
- The CAPE Admissions button is already connected to the current CAPE Google Apps Script portal.
- The Grade 7 Admissions page currently displays “Registration Portal Coming Soon”.
- When the Grade 7 Apps Script is deployed, open grade7-admissions.html and replace the coming-soon button href="#" with the new /exec URL. Also remove the aria-disabled and onclick attributes.
